Tecnologias utilizadas

HTML

HTML (HyperText Markup Language)

Site Oficial: BDJHBV

Definição da Tecnologia: HTML é a linguagem de marcação padrão usada para criar e estruturar páginas na web. Ele define a estrutura básica de conteúdo de um documento web usando uma série de elementos e tags.

Prós e Limitações

Prós
Simplicidade: Fácil de aprender e usar, ideal para iniciantes.
Compatibilidade: Compatível com todos os navegadores e plataformas.
SEO Friendly: Estrutura de documentos que ajuda na otimização para motores de busca.

Limitações
Estático: Por si só, HTML não oferece funcionalidades dinâmicas.
Segurança: O código HTML é visível a todos, o que pode expor vulnerabilidades.
Dependência de CSS e JavaScript: Para uma apresentação e funcionalidade completas, depende dessas outras tecnologias.

CSS

CSS (Cascading Style Sheets)

Site Oficial: BDJHBV

Definição da Tecnologia: CSS é uma linguagem de estilos usada para descrever a apresentação de um documento escrito em HTML ou XML. Ela controla o layout de múltiplas páginas da web ao mesmo tempo.

Prós e Limitações

Prós
Estilo e Design: Permite um controle detalhado sobre a aparência de uma página.
Reutilização: CSS externo pode ser reutilizado em várias páginas, economizando tempo e esforço.
Acessibilidade Melhora a acessibilidade ao separar a estrutura do conteúdo da apresentação.

Limitações
Compatibilidade entre Navegadores: Pode haver diferenças na renderização entre diferentes navegadores.
Complexidade: Pode se tornar complexo em grandes projetos, dificultando a manutenção.
Dependência: Funcionalidade limitada sem HTML e JavaScript.

JavaScript

JavaScript

Site Oficial: BDJHBV

Definição da Tecnologia: JavaScript é uma linguagem de programação que permite a criação de conteúdo dinâmico e interativo em páginas web. Ele pode manipular elementos HTML e CSS, além de fornecer funcionalidades avançadas.

Prós e Limitações

Prós
Interatividade: Adiciona interatividade e dinamismo às páginas web.
Versatilidade: Pode ser usado tanto no front-end quanto no back-end (Node.js).
Extensa Biblioteca:Grande variedade de bibliotecas e frameworks disponíveis (como React, Angular, e Vue).

Limitações
Segurança: Pode ser vulnerável a ataques como Cross-Site Scripting (XSS).
Desempenho: O uso excessivo pode afetar o desempenho da página.
Compatibilidade: Diferenças na implementação entre navegadores podem causar problemas.
